== Installation ==

= Minimum Requirements =

* WordPress 5.3 or greater
* PHP version 5.6 or greater
* MySQL version 5.6 or greater

= Installation =

1. Upload the `devmonsta` folder to the `/wp-content/plugins/` directory
1. Activate the devmonsta plugin through the 'Plugins' menu in WordPress
1. Configure the plugin by going to the `devmonsta` menu that appears in your admin menu
